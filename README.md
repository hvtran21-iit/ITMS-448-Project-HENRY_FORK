# better_Toone
A program that connects to your spotify and gives you a movie based on your recently listened songs.

--
To run:
1.) Make sure you have flask installed
2.) Run app.py either on VS Code, or through the terminal after navigating to the GitHub folder
3.) Search the given host from running app.py on your browser
